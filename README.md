# PDF処理

Python と PyPDF2 を使った PDF 操作用の GUI ツールです。  
GUI 上で簡単に PDF のページ抽出、結合、交互結合、逆順処理ができます。

---

## 機能

- **PDF抽出**  
  PDF から必要なページだけを取り出す。
  - ページ指定は `1,3-5,7` のように、カンマ区切りと範囲指定が可能です。

- **PDF結合**  
  複数（最大5）PDF を順番に結合して1つにまとめる。

- **PDF交互結合**  
  2つの PDF のページを交互に並べて1つにまとめる。

- **PDF逆順**  
  PDF のページ順を逆にして新しい PDF を作る。
